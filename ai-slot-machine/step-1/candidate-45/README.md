# AI Slot Machine

A simple and fun slot machine game that makes fun of AI, built with vanilla web technologies.

## How to Play

1.  Open the `index.html` file in your web browser.
2.  Click the "Spin" button to play.
3.  Each spin costs 10 "compute credits".
4.  Match three symbols in a row to win more credits.
5.  The game is over when you run out of credits.

## Technologies Used

*   HTML
*   CSS
*   JavaScript
*   Web Speech API

## About the Game

This slot machine was created to be a fun, lightweight game that pokes fun at the world of AI. The "compute credits" and AI-themed symbols are a nod to the resources and concepts that power modern artificial intelligence. The use of the Web Speech API adds an interactive element, with the game providing spoken feedback to the player.
