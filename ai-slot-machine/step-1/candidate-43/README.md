# AI Slot Machine

This is a simple slot machine game built with vanilla HTML, CSS, and JavaScript. The theme of the game is a fun take on AI, where you win and lose tokens.

## How to Play

1.  Open the `index.html` file in your web browser.
2.  Click the "Spin" button to play.
3.  Each spin costs 1 token.
4.  You win tokens by matching symbols on the reels.
5.  The game ends when you run out of tokens.

## Symbols and Payouts

*   **3 of a kind:**
    *   🤖: 50 tokens
    *   🧠: 40 tokens
    *   ⚡️: 30 tokens
    *   🔥: 20 tokens
    *   💻: 10 tokens
    *   💡: 5 tokens
    *   📈: 2 tokens
    *   🚀: 1 token
*   **2 of a kind:** 1/5th of the 3-of-a-kind value.

## Files

*   `index.html`: The main HTML file.
*   `style.css`: The CSS file for styling the game.
*   `script.js`: The JavaScript file for the game logic.
